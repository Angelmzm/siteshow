# SiteShow 🎉

O SiteShow é uma plataforma desenvolvida para centralizar e organizar eventos urbanos.

## 🚀 Funcionalidades
- Coleta de eventos via CSV
- Web Scraping de eventos
- Classificação automática
- Identificação de eventos urgentes

## 🛠️ Tecnologias
- Python
- Pandas
- Streamlit
